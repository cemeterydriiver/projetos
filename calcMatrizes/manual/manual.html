Como usar a Calculadora

  • Adicionando matrizes

    Para adicionar uma matriz clique no botão Adicionar matriz no canto superior esquerdo ou aperte Alt + Shift + A.

    Após clicar no botão vai aparecer uma caixa na tela pedindo o nome da matriz.

    Digite um nome e clique em "Ok".

    Você pode adicionar quantas matrizes você quiser.

  • Configurando as matrizes

    Linhas e Colunas

    Entre o nome da matriz e o menu de funções vão estar dois inputs.

    O primeiro input é a quantidade de linhas

    O segundo é a quantidade de colunas

    Para adicionar ou remover as linhas e colunas é só clicar nas setas dentro do input ou digitar o valor no input.

  • Menu de funções

    Limpar matriz

    Apaga todos os valores da matriz

    Copiar Valores

    Copia os valores da matriz

    Colar Valores

    Cola os valores que foram copiados

    Renomear matriz

    Abre um pop-up com uma caixa de texto para você colocar um novo nome na matriz

    Deletar matriz

    Deleta a matriz

  • Utilizando a barra de escrita

    A barra de escrita é o input de texto que fica em cima do botão Calcular.

    Nessa barra você escreve as funções que deseja usar para fazer os cálculos.

    Exemplo:

    Desejo descobrir o determinante de uma matriz

    Antes de fazer o cálculo eu tenho que por um nome na matriz. Neste exemplo vou nomear a matriz como matriz exemplo

    para fazer o cálculo eu escrevo det(matriz exemplo) e clico em Calcular ou aperto Enter

    Embaixo do botão há um espaço onde vai aparecer o resultado e se houver algum erro, ele vai aparecer abaixo da barra em vermelho.

    Utilizando mais de uma função nos cálculos

    Uma das qualidades desta calculadora de matrizes é possibilidade de fazer cálculos com mais de duas matrizes e funções.

    Exemplos:

    1. Desejo fazer a soma de três matrizes.

      Para isso usamos soma(nome da primeira matriz, nome da segunda matriz, nome da terceira matriz).

      A função de soma pode somar a quantidade de matrizes que você desejar ao mesmo tempo.

    2. Desejo fazer a multilplicação de uma matriz com a inversa de outra matriz.

      Para isso podemos usar o seguinte comando multi(Nome da primeira matriz, inv(Nome da segunda matriz)).

    3. Último exemplo:

      Desejo subtrair duas matrizes e com o resultado da operação quero extrair o determinante.

      Para isso podemos usar o seguinte comando det(sub(Nome da primeira matriz, Nome da segunda matriz))

    Regras:

    toda função tem na frente dois parênteses (), dentro desses parênteses vai o nome das matrizes que você deseja usar no cálculo ou um número (depende da função).

    Se por acaso você tiver que colocar mais de um elemento dentro dos parênteses você deve colocar uma vígula para separá-los, senão o programa não entende que existem mais de dois elementos dentro dos parênteses.

  • Deletando todas as matrizes

    Para deletar todas as matrizes é só clicar no botão Deletar todas as matrizes ao lado do botão Adicionar matriz ou aperte Alt + Shift + D.

Funções

  • soma

    Faz a soma das matrizes

    soma(nome_da_matriz, nome_da_matriz, nome_da_matriz)

  • sub

    Faz a subtração das matrizes

    sub(nome_da_matriz, nome_da_matriz, nome_da_matriz)

  • multi

    Faz a multiplicação das matrizes

    multi(nome_da_matriz, nome_da_matriz)

  • n

    Multiplica os valores da matriz por um número

    n(nome_da_matriz, número)

  • det

    Calcula o determinante da matriz

    det(nome_da_matriz)

  • oposta

    Calcula a matriz oposta

    oposta(nome_da_matriz)

  • id

    Gera uma matriz identidade na ordem do número dentro dos parâmetros

    id(número)

  • trans

    Calcula a matriz transposta

    trans(nome_da_matriz)

  • adj

    Calcula a matriz Adjunta

    adj(nome_da_matriz)

  • inv

    Calcula a matriz inversa

    inv(nome_da_matriz)

  • colunas

    Representa a quantidade de colunas da matriz

    colunas(nome_da_matriz)

  • linhas

    Representa a quantidade de linhas da matriz

    linhas(nome_da_matriz)

Regras para o nome das matrizes

  1. Pode ser usado qualquer tipo de caractere no nome das matrizes, mas o nome não deve ser composto apenas por caracteres especiais, pois o programa não considera eles.

  2. Não pode haver duas matrizes com o mesmo nome.

  3. O nome das matrizes não pode ser iguais aos nomes das funções do programa.

Atalhos do teclado

  • Alt + Shift + A

    Adiciona uma nova matriz

  • Alt + Shift + D

    Remove todas as matrizes

  • Alt + Shift + L

    Limpa a caixa de resultado

  • Enter

    Faz o cálculo
